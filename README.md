# 📋 Sobre o projeto

Este sistema nasceu da necessidade de ter um ambiente limpo, rápido e confiável para preencher, visualizar e exportar fichas cadastrais.

Ele funciona inteiramente no navegador: você digita os dados nos campos à esquerda e, instantaneamente, a ficha à direita é atualizada — como se estivesse preenchendo um formulário de papel, mas com a praticidade do digital.

O layout é pensado para ser funcional e ao mesmo tempo familiar. 

Ah, e tem um detalhe especial: **o Clippy está de volta.**  
Ele aparece sempre com uma frase inspiradora — só para lembrar que, mesmo no trabalho, um toque de leveza faz bem.

# ✨ Principais características

## 🖊️ Preenchimento em tempo real
Enquanto você digita nos campos do painel esquerdo, a ficha à direita é atualizada automaticamente.  
Tudo visível, sem surpresas.

## 📁 Carregamento a partir de arquivo
Se você já tem os dados em um arquivo de texto no formato esperado, basta usar o botão com ícone de pasta (📁) e carregar o arquivo.  
Os campos serão preenchidos automaticamente.

## ✅ Validação amigável
Campos como **CPF, matrícula e CEP** contam com formatação automática e dicas visuais caso algo esteja fora do padrão.

Nada de mensagens agressivas — apenas pequenos lembretes.

## 📅 Controle sobre datas opcionais
As seções de **aposentadoria** e **desligamento** podem ser ativadas ou desativadas com um simples checkbox.

Quando desativadas, os campos correspondentes somem da ficha, mantendo tudo organizado.

## 🔒 Privacidade total
Todos os dados permanecem **no seu computador**.

Nenhuma informação é enviada para a internet ou armazenada em servidores.

Você pode fechar a página tranquilo.

## 📱 Design responsivo
A interface se adapta a telas menores.

Ao **imprimir**, a ficha ganha uma versão limpa, sem os controles — ideal para arquivamento físico.

# 🔍 Como funciona na prática

Acesse:

https://brunomoraesdigital.github.io/ficha-cadastral/

Você tem duas maneiras de usar:

### ✍️ Manual
Preencha cada campo com as informações necessárias.  
A ficha reage a cada digitação.

### ⚡ Automática
Clique no botão de pasta, selecione um arquivo `.txt` com os dados no formato correto e pronto — tudo preenchido em segundos.

# 📑 Informações exibidas na ficha

A ficha organiza os dados em blocos visuais claros:

- **Dados pessoais**  
  Nome, CPF, matrícula, filiação, naturalidade etc.

- **Informações funcionais**  
  Cargo, carreira, vínculo, carga horária, lotação.

- **Datas importantes**  
  Admissão, aposentadoria e desligamento.

- **Contato**  
  Telefones e e-mail.

- **Endereço completo**

Tudo estruturado para facilitar **leitura e conferência**.
